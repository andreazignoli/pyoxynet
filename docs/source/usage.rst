Usage
=====

Installation
------------

To use pyoxynet, first install it using pip:

.. code-block:: console

   (.venv) $ pip install pyoxynet


PrintHello
----------------

To test the package you can use the ``pyoxynet.PrintHello()`` function:

.. py:function:: pyoxynet.PrintHello()

    Parameters
    ----------
    nothing : str
        Nothing at all
    nothing_at_all : bool, optional
        A flag not used to do anything (default is False)

    Returns
    -------
    str
        A str saying Hello

To test the package you can use the ``utilities.PrintHello()`` function:

.. autofunction:: utilities.PrintHello()
